== Description ==

Tisara is premium jewelry WooCommerce Theme from Tokomoo, help you building your jewelry store quickly with powerful WooCommerce WordPress plugin.

For more information about Tisara Jewelry WooCommerce Theme please go to 

https://themeforest.net/user/tokomoo/portfolio

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Upload both tisara-wp.zip (parent theme) and tisara-wp-child.zip (child theme)
3. Activate Tisara Child Theme to start using Tisara
4. Go to https://tokomoo.helpscoutdocs.com/ for a guide to customize this theme.
5. Navigate to Appearance > Customize in your admin panel.
